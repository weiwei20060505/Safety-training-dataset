📊 評估指標說明
================================================================================

對於每個模型，計算 5 個指標：

1. Accuracy（準確率）
   └─ 定義：(TP + TN) / (TP + TN + FP + FN)
   └─ 含義：整體預測正確率

2. Precision（精確率）
   └─ 定義：TP / (TP + FP)
   └─ 含義：預測為正的樣本中，實際為正的比例

3. Recall（召回率）
   └─ 定義：TP / (TP + FN)
   └─ 含義：實際為正的樣本中，被正確預測的比例

4. F1 Score
   └─ 定義：2 × (Precision × Recall) / (Precision + Recall)
   └─ 含義：Precision 和 Recall 的調和平均數

5. AUC (Area Under Curve)
   └─ 定義：ROC 曲線下的面積
   └─ 含義：模型區分正負樣本的能力（0.5 = 隨機，1.0 = 完美）

================================================================================
🔍 兩個預測任務說明
================================================================================

Y1 (提示詞有害性)
────────────────
- 目標：預測原始提示詞是否有害
- 標籤來源：data_type 欄位 (harmful=1, benign=0)
- 應用：檢測輸入提示詞的安全性

Y3 (一致性)
──────────
- 目標：預測提示詞類型和模型回應是否一致
- 定義：y3 = (y1 == y2)
  - y1 = harmful (1) 或 benign (0)
  - y2 = unsafe (1) 或 safe (0)
  - 一致 (1)：harmful -> unsafe 或 benign -> safe
  - 不一致 (0)：harmful -> safe 或 benign -> unsafe
- 應用：檢測模型判斷的一致性
